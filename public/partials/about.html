About


Med List is an app that allows users to keep track of medications by creating med lists. Lists contain medication name, along with pertinent information about the medication such as dosage, directions for use, and special notes. You can create multiple lists to suit your needs.

To get started, click Add New List to create a new list. After you've created a list, click the name of the list to start adding medications. Click the Add Medication button to enter the medication details.

You can print your medication list by selecting the Print button. You can also search a list using the search bar located at the top of each list.

To edit the name of a list, from the main page, click for the name of the list you'd like to edit.
To delete a list, from the main page, click for the list you would like to delete. This will delete the list and all of the medications associated with that list.
